# UI Kit - Vue 2 + Quasar + Webpack 4

Демонстрационный проект UI Kit, построенный на Vue 2, Quasar Framework 1.x и Webpack 4.

## Технологии

- **Vue.js 2.6.14** - прогрессивный JavaScript фреймворк
- **Vuex 3.6.2** - управление состоянием приложения
- **Quasar Framework 1.16.9** - Material Design компоненты
- **Webpack 4.46.0** - сборщик модулей
- **Babel** - транспиляция современного JavaScript

## Структура проекта

```
ui-kit/
├── public/
│   └── index.html          # Главный HTML файл
├── src/
│   ├── components/         # Vue компоненты
│   │   ├── ButtonsDemo.vue
│   │   ├── FormsDemo.vue
│   │   ├── CardsDemo.vue
│   │   └── DialogsDemo.vue
│   ├── store/              # Vuex store
│   │   └── index.js
│   ├── App.vue             # Главный компонент
│   └── main.js             # Точка входа
├── package.json            # Зависимости и скрипты
├── webpack.config.js       # Конфигурация Webpack
├── .babelrc               # Конфигурация Babel
└── README.md              # Документация
```

## Установка и запуск

### Установка зависимостей

```bash
npm install
```

### Разработка

```bash
npm run dev
```

Приложение будет доступно по адресу: http://localhost:8080

### Сборка для продакшена

```bash
npm run build
```

### Сборка для разработки

```bash
npm run build:dev
```
